# Permissions

## Wildcard

- `ezworldflags.*` → grants all EzWorldFlags permissions.

## Individual nodes

- `ezworldflags.gui` → open the world selector and world GUI.
- `ezworldflags.set` → use `/worldflags set ...`.
- `ezworldflags.get` → use `/worldflags get ...`.
- `ezworldflags.list` → use `/worldflags list ...`.
- `ezworldflags.reload` → use `/worldflags reload`.
- `ezworldflags.reset` → use `/worldflags reset ...`.

By default these are operator-level permissions.
