# EzWorldFlags Documentation

Welcome to the official user documentation for **EzWorldFlags**.

EzWorldFlags is a lightweight Paper/Folia plugin that lets you control gameplay behavior **per world** using a simple flag system:

- `ALLOW` → explicitly allow behavior in that world.
- `DENY` → explicitly block behavior in that world.
- `DEFAULT` → inherit from your server-wide defaults in `config.yml`.

## Quick Start

1. Install the jar into `plugins/`.
2. Start the server once.
3. Run `/worldflags` in-game as an operator.
4. Select a world and configure flags.
5. Use **Save & Close** in the GUI.

If you prefer command-line administration, jump to [Commands](commands.md).
